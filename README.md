# Smart Intruder Alarm System

## Overview
This project uses a PIR Sensor with Arduino UNO to detect motion and trigger an alarm.

## Components Used
- Arduino UNO
- PIR Sensor
- LED
- Buzzer
- Resistor

## Working
When motion is detected, LED and buzzer turn ON. When no motion is detected, both remain OFF.

## Applications
- Home Security
- Office Security
- Restricted Area Monitoring

## Tools Used
- Arduino IDE
- Wokwi Simulator
